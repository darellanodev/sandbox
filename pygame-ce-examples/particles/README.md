# Particles

A mini example of using particles in `pygame community edition` <https://pyga.me/>

## Installation

This example was made in Python 3.14.0 and pygame-ce 2.5.6 with VSCode

- Create the virtual environment with `python -m venv .venv`
- Activate the environment with VSCode or execute in git bash terminal with `source .venv/Scripts/activate`
- Upgrade pip with `python -m pip install --upgrade pip`
- Install the dependencies with `python -m pip install -r requirements.txt`

## Start the application

- Execute `python main.py`
- Move the mouse while the particles appears. Click to stop the creation of particles.
